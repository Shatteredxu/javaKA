##### 1.链表找中点技巧(双指针)

如果链表⻓度为偶数，也就是说中点有两个的时候，我们这个解法返回的节点是靠后的那个节点。

```java
ListNode middleNode(ListNode head) {
 // 快慢指针初始化指向 head
   ListNode slow = head, fast = head;
   // 快指针⾛到末尾时停⽌
   while (fast != null && fast.next != null) {
   // 慢指针⾛⼀步，快指针⾛两步
     slow = slow.next;
     fast = fast.next.next;
   }
   // 慢指针指向中点
   return slow;
}
```
