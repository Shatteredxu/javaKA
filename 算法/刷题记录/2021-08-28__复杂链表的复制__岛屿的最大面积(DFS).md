#### [剑指 Offer 35. 复杂链表的复制](https://leetcode-cn.com/problems/fu-za-lian-biao-de-fu-zhi-lcof/)

```java
时空复杂度都为O(n)
class Solution {
    public Node copyRandomList(Node head) {
        if(head==null)return head;
        Map<Node,Node> map = new HashMap<>();
        for(Node cur = head;cur!=null;cur = cur.next){
            Node newNode = new Node(cur.val);
            map.put(cur,newNode);
        }
        for(Node cur = head;cur!=null;cur = cur.next){
            map.get(cur).next = map.get(cur.next);
            map.get(cur).random = map.get(cur.random);
        }
        return map.get(head);
    }
}
```

```java
class Solution {
    public Node copyRandomList(Node head) {
        if(head==null)return head;
        for(Node cur = head;cur!=null;cur = cur.next.next){//将复杂链表复制两份
            Node node = new Node(cur.val);
            node.next = cur.next;
            cur.next = node;
        }
        for(Node cur = head;cur!=null;cur = cur.next.next){//将random指针进行拷贝
            if(cur.random!=null){
                cur.next.random = cur.random.next;
            }
        }
        Node cur = head;
        Node copyNode = head.next;
        Node copyNext = copyNode;//将链表分开，然后返回
        for(;cur!=null;cur = cur.next,copyNext = copyNext.next){
            cur.next = cur.next.next;
            if(copyNext.next!=null){
                copyNext.next = copyNext.next.next;
            }
        }
        return copyNode;
    }
}
```

### [695. 岛屿的最大面积](https://leetcode-cn.com/problems/max-area-of-island/)

##### 深度优先遍历

```java
class Solution {
    int res = 0;
    public int maxAreaOfIsland(int[][] grid) {
        int a = grid.length;
        int b = grid[0].length;
        for(int i = 0;i<a;i++){
            for(int j = 0;j<b;j++){
                if(grid[i][j]==1){
                    dfs(grid,i,j);
                }
                
            }
        }
        return res;
    }
    int dfs(int[][] grid,int i,int j ){
        if(i<0||i>=grid.length||j<0||j>=grid[0].length||grid[i][j]==0)return 0;
        grid[i][j]=0;
        int count = 0;
        count+=(1+dfs(grid,i-1,j)+dfs(grid,i+1,j)+dfs(grid,i,j-1)+dfs(grid,i,j+1));
        res  = Math.max(count,res);
        return count;
    }
}
```

##### 广度优先遍历

```Java
class Solution {
    int res = 0;
    public int maxAreaOfIsland(int[][] grid) {
        int a = grid.length;
        int b = grid[0].length;
        for(int i = 0;i<a;i++){
            for(int j = 0;j<b;j++){
                if(grid[i][j]==1){
                    res = Math.max(dfs(grid,i,j),res);
                }
            }
        }
        return res;
    }
    int dfs(int[][] grid,int i,int j ){
        if(i<0||i>=grid.length||j<0||j>=grid[0].length||grid[i][j]==0)return 0;
        Queue<int[]> queue = new LinkedList<>();
        queue.offer(new int[]{i,j});
        grid[i][j] = 0;
        int count = 1;
        while(!queue.isEmpty()){
            int[] tmp = queue.poll();
            int[] X = {1, -1, 0, 0};
            int[] Y = {0, 0, 1, -1};
            for(int k = 0;k<4;k++){
               int newX = tmp[0]+X[k];
               int newY = tmp[1]+Y[k];
               if(newX>=0&&newX<grid.length&&newY>=0&&newY<grid[0].length&&grid[newX][newY]==1){
                   queue.offer(new int[]{newX,newY});
                   count+=1;
                   grid[newX][newY] = 0;
               }
            }
        }
        return count;
    }
}
```
