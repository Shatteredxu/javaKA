### [112. 路径总和](https://leetcode-cn.com/problems/path-sum/)

递归

```java
class Solution {
    public boolean hasPathSum(TreeNode root, int targetSum) {
        return backTrack(root,targetSum);
    }
    boolean backTrack(TreeNode root, int targetSum){
        if(root==null){return false;}//不能判断target<0因为没说一定为正数
        if(root.left==null&&root.right==null&&targetSum-root.val == 0){
            return true;
        }
        return backTrack(root.left,targetSum-root.val)||backTrack(root.right,targetSum-root.val);
    }
}
```

### [143. 重排链表](https://leetcode-cn.com/problems/reorder-list/)

使用双端队列的做法

```java
class Solution {
    public void reorderList(ListNode head) {
        ListNode q = head;
        ListNode p = head;
        Deque<ListNode> deque = new LinkedList();
        while(q!=null){
            deque.add(q);
            q = q.next;
        }
        deque.removeFirst();
        while(deque.size()>=2){
            p.next = deque.getLast();
            deque.removeLast();
            p=p.next;
            p.next = deque.getFirst();
            deque.removeFirst();
            p=p.next;
        }
        if(deque.size()==1){ p.next = deque.getFirst();
            deque.removeFirst();p=p.next;}
        p.next = null;
    }
}
```

使用线性表存储，，每次可以按下表取出对应节点

```java
class Solution {
    public void reorderList(ListNode head) {
        ListNode q = head;
        ListNode p = head;
        List<ListNode> list = new ArrayList<ListNode>();
        while(q!=null){
            list.add(q);
            q = q.next;
        }
       int i=1;
       int j = list.size()-1;
       while(i<=j){//第一个位置p已经在链表中，不需要取出
           p.next = list.get(j);
           p = p.next;
           j--;
           if(i>j){break;}
           p.next = list.get(i);
           p = p.next;
           i++;
       }
       p.next = null;
    }
}
```

原地重排，使用快慢指针先找到中点，然后再翻转后半部分的链表，再进行相连

```java
class Solution {
    public void reorderList(ListNode head) {
        if(head==null||head.next == null)return;
        ListNode fast = head;
        ListNode slow = head;
        while(fast!=null&&fast.next!=null){
          //如果是偶数，能找到后半部分的第一个
          //如果是奇数，能找到最中间的那一个
            fast = fast.next.next;
            slow = slow.next;
        }
        ListNode needReverse = slow.next;
        slow.next = null;
        ListNode p = needReverse;
        ListNode r = null;
        while(p!=null){
            ListNode node = p.next;
            p.next = r;
            r = p;
            p = node;
        }
        ListNode cur = head;
      //将两个链表进行相连
        while(cur!=null&&r!=null){
            ListNode nextCur = cur.next;
            ListNode nextr = r.next;
            cur.next = r;
            r.next = nextCur;
            cur = nextCur;
            r = nextr;
        }
    }
}
```

### [109. 有序链表转换二叉搜索树](https://leetcode-cn.com/problems/convert-sorted-list-to-binary-search-tree/)

```java
class Solution {
    public TreeNode sortedListToBST(ListNode head) {
        if(head==null)return null;
        else if(head.next==null)return new TreeNode(head.val);
        ListNode fast = head;//快慢指针找中点
        ListNode slow = head;
        ListNode pre =head;//找到中点前面的那个节点，
        while(fast!=null&&fast.next!=null){
            pre = slow;
            fast = fast.next.next;
            slow = slow.next;
        }
        pre.next = null;//将两条链表分开
        TreeNode root= new TreeNode(slow.val);
        root.left = sortedListToBST(head);
        root.right = sortedListToBST(slow.next);
        return root;
    }
}
```
